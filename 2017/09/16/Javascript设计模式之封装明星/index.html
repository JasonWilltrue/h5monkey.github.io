问题：

通过this添加的属性和方法与通过prototype添加的属性和方法有什么区别？

通过this添加时在当前对象上添加的，
每创建一个对象。它都有一个原型prototype指向其继承的属性方法，这样通过prototype继承的方法并不是自身，this的属性和方法是自身拥有，每次通过类创建一个新对象指向自身属性和方法
而通过prototype继承通过类创建属性和方法不会再次创建

constructor是什么？

在prototype对象中会出现一个和函数创建this一样的一个constructor属性
constructor属性指向就是拥有这个原型对象的函数或对象
