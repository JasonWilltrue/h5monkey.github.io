这篇文章的目标明确，我们要实现以下两个功能：

  1. 异步更新 state，将短时间内的多个 setState 合并成一个
  2. 为了解决异步更新导致的问题，增加另一种形式的 setState：接受一个函数作为参数，在函数中可以得到前一个状态并返回下一个状态
