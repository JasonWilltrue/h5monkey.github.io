很早之前一直想读一下jquery或者zepto的源码，但读源码之前还是得先了解下上面是链模式；因为源码的框架都是基于这个链模式

链模式：通过对象方法中将当前对象返回，实现同一个对象多个方法的链式调用，从而简化该对象多个方法的多次调用，对该对象的多次引用。
