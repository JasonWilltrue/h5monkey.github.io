怎么实现 this._test 改变而不是 this._data.test 改变触发更新？

数据代理: 通过一个对象代理对另一个对象(在前一个对象内部)中属性的操作(读/写)

vue 数据代理: 通过 vm 对象来代理 data 对象中所有属性的操作

好处: 更方便的操作 data 中的数据

基本实现流程

  1. a. 通过 Object.defineProperty()给 vm 添加与 data 对象的属性对应的属性描述符
  2. b. 所有添加的属性都包含 getter/setter
  3. c. getter/setter 内部去操作 data 中对应的属性数据
