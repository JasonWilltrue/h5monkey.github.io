Vue中的slot

  • Vue

在 2.6.0 中，我们为具名插槽和作用域插槽引入了一个新的统一的语法 (即 v-slot 指令)。它取代了 slot 和 slot-scope 这两个目前已被废弃但未被移除

Slot的通俗理解是“占坑”，在组件模板中占好了位置，当使用该组件标签时候，组件标签里面的内容就会自动填坑（替换组件模板中位置），当插槽也就是坑有命名时，组件标签中使用属性slot=”mySlot”的元素就会替换该对应位置内容；

单个插槽

看单个插槽，即没有在标签中显式的设置name值。比如SlotDemo这个组件，而且在slot标签中提供了一个默认的内容。我们在使用该组件时，可以通过v-slot:default来调用未显式设置name的slot：一个不带name的出口会带有隐含的名字default
